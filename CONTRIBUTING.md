# Contributing

Pull requests are welcome.

## Setup

1. ```git clone https://github.com/mlajtos/L1.git```
1. ```cd L1```
1. ```yarn```

## Development

1. ```yarn run dev```
1. http://localhost:7171

## Build

1. ```yarn run build```
1. ```yarn run serve```
1. http://localhost:7171

Directory `L1/dist` contains built project.
